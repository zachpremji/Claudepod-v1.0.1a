# ClaudePod v0.1.0-alpha

Unified MCP Command Hub & Reference — 57 MCP connectors, 5 platform chains, 280+ execution chains.

## Quick Start

```bash
npm install
npm run dev
```

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion (chain animations)
- Zustand (state management)
- Vitest (testing)

## Running Tests

```bash
npm test
```
